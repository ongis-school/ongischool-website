Apa itu Ongischool?

Ongis.School adalah sebuah program dari Digital Innovation Lounge (DILo) Malang Bersama Komunitas IT di Kota Malang. Program ini bertujuan mengembangkan Skill dan kemampuan programmer yang ada di Malang melalui pelatihan dan workshop Teknis. Dalam program ini terdapat 4 Komunitas yang akan bersinergi dengan DILo Malang, yakni Malang PHP User Group, Malang Frontend Developer, Malang-JS (Java Script) dan KelasMobile Malang.

Frequently Ask Question Program Ongis.School
Kapan Program ini akan dimulai ?

Program Ongis.School ini akan dimulai Bulan April sampai Juni 2016. Program ini akan terlaksana 2 kali dalam sebulan (Masing-masing kelas).

Dimana Program ini akan dilaksanakan & Siapa Mentornya ?

Program Ongis.School dilaksanakan di Digital Innovation Lounge Malang (DILo) Jl. Basuki rachmad 7-9 Malang (Telkom Kayutangan). Mentor Program ini adalah para expert dari masing-masing Komunitas IT.

Apa yang bisa didapat dalam Program ini?

Tentunya kita mendapat ilmu seputar dunia programming dari para Expert. selain itu Students juga mendapatkan Sertifikat di akhir pertemuan kelas jika dinyatakan Lulus.

Bagaimana Syarat lulus dari Program ini?

Mengikuti Program ini dengan baik dan seksama. Batas toleransi dari penyelenggara adalah 2 kali absen (tidak ikut). jika melebihi 2 Kali pertemuan tidak masuk dianggap Tidak Lulus dan Uang Komitmen Fee akan Hangus
